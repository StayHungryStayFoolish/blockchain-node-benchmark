# Architecture Decision Records (ADR)

每当 NORTH-STAR.md 决策表发生变更时,本目录新增一份 ADR 记录决策背景。

## ADR 文件命名

`NNNN-<short-name>.md`,NNNN 为 4 位编号(0001 起)

## ADR 模板

```markdown
# ADR-NNNN: <决策标题>

## 状态
Proposed / Accepted / Deprecated / Superseded by ADR-XXXX

## 背景
为什么要做这个决策?触发条件是什么?

## 选项
- 选项 A:...
- 选项 B:...
- 选项 C:...

## 决策
选定方案 + 一句话理由

## 后果
- 正面影响
- 负面影响
- 后续工作
```
