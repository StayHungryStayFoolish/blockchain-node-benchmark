Production Environment Recommendation: Total monitoring overhead usually accounts for 1-3% of system resources and can be ignored.

Reasons for IOPS/Throughput Being 0:

  • Monitoring system mainly reads /proc virtual filesystem, kernel does not count physical I/O statistics
  • Actual I/O overhead < 0.00005 IOPS/s, even with 4 decimal precision (%.4f) still shows as 0.0000
  • This proves the monitoring system is efficiently designed with almost no impact on production environment
  • To view tiny values, increase precision to %.6f or higher in source code
